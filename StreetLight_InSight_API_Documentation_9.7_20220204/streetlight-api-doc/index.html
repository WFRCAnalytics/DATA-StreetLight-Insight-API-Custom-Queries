StreetLight InSight® Data API Technical Documentation

This folder contains the technical documentation for the StreetLight InSight Data API. The Data API provides a programmatic interface that is equivalent to the StreetLight InSight application that is accessible via the web interface. For more information about StreetLight InSight, please visit streetlightdata.com.

Before using the Data API, we recommend that you visit the Support Center and watch the training webinar for a product overview of StreetLight InSight.

The documentation in this folder is organized in the following sections:

  • Quick Start tutorial

  • API Reference

  • Client Code Samples
